# /meeting - 会议管理

记录和管理会议

## 使用方式

```
/meeting new [时间] [议题]    # 新建会议
/meeting list                 # 查看会议列表
/meeting note [会议ID]        # 记录会议纪要
```

## 新建会议

```markdown
# 会议安排

- **时间**: YYYY-MM-DD HH:MM
- **议题**: [议题]
- **参会人**: [待确认]
- **会议ID**: MTG-YYYYMMDD-01

已添加到会议列表。需要我提前提醒你吗？
```

## 会议纪要模板

```markdown
# 会议纪要 - MTG-YYYYMMDD-XX

## 基本信息
- **时间**: YYYY-MM-DD HH:MM
- **参会人**: [列表]
- **记录人**: 麦克斯

## 议题讨论

### 议题1: [标题]
**讨论内容**:
[要点记录]

**结论**:
[决定事项]

**待办**:
- [ ] [事项] - [负责人] - [截止时间]

### 议题2: [标题]
...

## 决议汇总
| 决议 | 负责人 | 截止时间 |
|------|--------|----------|
| [决议] | [人] | [时间] |

## 下次会议
- **时间**: [待定]
- **预定议题**: [议题]
```

## 存储位置

会议记录存储到 `../shared/tasks/meetings.md`
