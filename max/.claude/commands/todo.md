# /todo - 待办事项管理

管理个人和项目待办事项

## 使用方式

```
/todo                      # 查看所有待办
/todo add [内容]           # 添加待办
/todo done [ID]            # 完成待办
/todo project [项目名]     # 查看项目待办
```

## 待办格式

```markdown
# 待办事项

## 个人事务
- [ ] #P001 [事项] - 截止: [日期]
- [x] #P002 [已完成事项] - 完成: [日期]

## 项目: [项目名]
- [ ] #T001 [任务] - 负责人: [人] - 截止: [日期]
- [ ] #T002 [任务] - 负责人: [人] - 截止: [日期]

## 今日重点
1. [最重要的事项]
2. [次重要的事项]
```

## 优先级标记

| 标记 | 含义 |
|------|------|
| 🔴 | 紧急重要 |
| 🟡 | 重要不紧急 |
| 🔵 | 紧急不重要 |
| ⚪ | 不紧急不重要 |

## 查看待办

```markdown
# 待办事项 - YYYY-MM-DD

## 今日待办 (X项)
- [ ] 🔴 #P001 [紧急事项]
- [ ] 🟡 #T003 [重要事项]

## 本周待办 (X项)
- [ ] #T004 [任务] - 截止: 周五

## 已逾期 (X项)
- [ ] ⚠️ #T002 [逾期任务] - 原定: [日期]
```

## 存储位置

待办事项存储到 `../shared/tasks/todos.md`
